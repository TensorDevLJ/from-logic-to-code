# 02. Sort Array of 0s, 1s, and 2s (Dutch National Flag)

**LeetCode:** https://leetcode.com/problems/sort-colors/
**Difficulty:** 🟡 Medium | **Pattern:** Dutch National Flag / Three Pointer

---

## 🧠 Intuition

**In Simple Words:**
Sort array containing only 0, 1, 2 without using sort function.
[2,0,2,1,1,0] → [0,0,1,1,2,2]

**Real-Life Analogy:**
Sorting Dutch national flag colors: Red (0), White (1), Blue (2). Partition into three sections.

**Core Idea (DNF Algorithm):**
Maintain 3 pointers: lo, mid, hi
- arr[0..lo-1] = all 0s
- arr[lo..mid-1] = all 1s
- arr[hi+1..n-1] = all 2s
- arr[mid..hi] = unsorted region

Process mid from left to right:
- If arr[mid] = 0: swap with arr[lo], lo++, mid++
- If arr[mid] = 1: mid++ (it's in right place)
- If arr[mid] = 2: swap with arr[hi], hi-- (DON'T increment mid — need to check new arr[mid])

---

## 🐢 Brute Force: O(N log N)

```java
// Just use Arrays.sort — but this is O(N log N), interviewer wants better
Arrays.sort(arr);
```

## ⚡ Better: Count Sort O(N) — 2 passes

```java
public static void sortColorsCounting(int[] arr) {
    int count0=0, count1=0, count2=0;
    for (int x : arr) {
        if (x==0) count0++;
        else if (x==1) count1++;
        else count2++;
    }
    int i = 0;
    while (count0-- > 0) arr[i++] = 0;
    while (count1-- > 0) arr[i++] = 1;
    while (count2-- > 0) arr[i++] = 2;
}
```

**Time:** O(N) — 2 passes | **Space:** O(1)

## 🚀 Optimal: Dutch National Flag — 1 pass!

```java
public class SortColors {
    public static void sortColors(int[] arr) {
        int lo = 0;           // Next position for 0
        int mid = 0;          // Current element
        int hi = arr.length - 1; // Next position for 2

        while (mid <= hi) {
            if (arr[mid] == 0) {
                // Swap arr[lo] and arr[mid]
                int temp = arr[lo];
                arr[lo] = arr[mid];
                arr[mid] = temp;
                lo++;
                mid++;
            } else if (arr[mid] == 1) {
                mid++;  // 1 is in correct region, just move forward
            } else {  // arr[mid] == 2
                // Swap arr[mid] and arr[hi]
                int temp = arr[mid];
                arr[mid] = arr[hi];
                arr[hi] = temp;
                hi--;
                // DON'T increment mid! New arr[mid] (from hi) is unknown
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {2, 0, 2, 1, 1, 0};
        sortColors(arr);
        System.out.println(Arrays.toString(arr)); // [0, 0, 1, 1, 2, 2]
    }
}
```

**Time:** O(N) — single pass! | **Space:** O(1)

**Dry Run:** [2,0,2,1,1,0], lo=0, mid=0, hi=5
```
mid=0: arr[0]=2 → swap(arr[0],arr[5]) → [0,0,2,1,1,2], hi=4. arr=[0,0,2,1,1,2]
mid=0: arr[0]=0 → swap(arr[0],arr[0]) → lo=1, mid=1
mid=1: arr[1]=0 → swap(arr[1],arr[1]) → lo=2, mid=2
mid=2: arr[2]=2 → swap(arr[2],arr[4]) → [0,0,1,1,2,2], hi=3
mid=2: arr[2]=1 → mid=3
mid=3: arr[3]=1 → mid=4
mid=4 > hi=3 → STOP
Result: [0,0,1,1,2,2] ✅
```

## 🔁 Pattern Recognition

**Pattern:** Dutch National Flag (3-way partition)
- Used in 3-way QuickSort (Dijkstra's algorithm)
- Any "partition into 3 groups" problem

---

## 🧠 How to Think (Interview View)

1. First thought: Sort → O(N log N)
2. "Only 3 values!" → Count them → 2 pass O(N)
3. "Can you do 1 pass?" → DNF with 3 pointers!

**Key insight:** Don't increment mid when arr[mid]==2 because the swapped value from hi is unknown!

---

## ⚠️ Common Mistakes

1. ❌ Incrementing mid when arr[mid]==2 (skips checking swapped element)
2. ❌ Wrong pointer initialization (hi = n-1 not n)
3. ❌ Loop condition mid < hi (should be mid <= hi)

---

## 🎯 Final Summary

- Dutch National Flag: lo=0, mid=0, hi=n-1
- arr[mid]=0: swap with lo, move both pointers
- arr[mid]=1: only move mid
- arr[mid]=2: swap with hi, move only hi (NOT mid!)
- O(N) time, O(1) space, single pass — optimal!
